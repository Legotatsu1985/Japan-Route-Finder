# WHAT IS IT?
This application is for searching flight routes of Japan that is intended for your flight simulation.
<br>Just type your origin and destination airports, then you can find routes as you wish.

# HOW TO USE?
## Setting your SWIM account
### If you are first time use of **SWIM**;
1. Go to [SWIM (System-wide information management)](https://web.swim.mlit.go.jp) (**Account registration is required.**) and login it.
2. Click **Service** in the top of the page.
3. Find **AIP File Download Service**, click it and open service details.
4. Click **Start Service**, and continue to the next process below.

### If you are already using **SWIM** and **"AIP File Download Service"**;
1. Go to [SWIM (System-wide information management)](https://web.swim.mlit.go.jp) and login it.
2. Click **In-Use Services** in the top of the page.
3. Click **AIP File Download Service**. (New tab will be opened.)
4. Click **2_AIC** folder.
5. Scroll-down and find the latest effective Excel file that is named like **CHANGE_OF_FLIGHT_PLANNED_ROUTES**.
6. Download the file and continue to the next process below.

## Software-side
1. Download our app from [here](https://github.com/Legotatsu1985/Japan-Route-Finder/releases).
2. Unzip the file and just launch the jar file.
3. Click **File** menu and open **Settings**.
4. Click **Browse** button and select the Excel file you downloaded in the previous process.
5. Make sure the path is correct, then click **Save**.
6. Type your origin and destination airports in the fields, then click **Search** button. The routes will be shown below.
